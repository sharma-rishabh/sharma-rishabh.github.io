Basics of Kafka

Kafka is a distributed event logging system. It’s widely used in event event driven system as framework of choice. Events We learned about these the last time in Event Driven Systems. In kafka an event is represented by a key-value pair. A key can be null as well. Topics A topic is an ordered log A new event is appended to end of a topic. A topic is not a queue it’s a proper Log.
